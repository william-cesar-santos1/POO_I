Crie uma calculadora de salário bruto.

O salário deve ser calculado com base na quantidade de horas trabalhadas. 
Ignore feriados e finais de semana, não existe hora extra.

O calculo deve ser feito da seguinte forma:
  total = horas_trabalhadas * valor_hora

O sistema deve pedir ao usuário que seja informado os registros de ponto para 
cada dia. Exemplo:
```
 Informe o valor da hora: 20.00  
 Informe a data: 01/03/2024  
 Informe a hora de entrada: 08:00  
 Informe a hora de saída: 12:00  
 Informe a hora de entrada: 13:00  
 Informe a hora de saída: 17:00  
 Deseja informar outro dia? (s/n): s
```  
Resultado:
```
 Foram trabalhadas 40 horas no total e o seu salário bruto é R$ 800.00  
```

O sistema deve permitir que o usuário informe os registros de ponto 
para vários dias.
